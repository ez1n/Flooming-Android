# Flooming-Android

딥러닝 모델을 활용한 꽃 사진 분류, 꽃 그림 생성 모바일 앱

## Framework
```React Native```, ```expo-cli```

## API / LIBRARY
```axios```, ```React Navigation```

## Functions
<ul>
  <li>꽃 사진 업로드 - 갤러리 사진, 촬영</li>
  <li>꽃 분류 (28개의 서로다른 꽃 클래스)</li>
  <li>꽃 사진 그림으로 변환</li>
  <li>꽃 그림 comment와 함께 업로드</li>
  <li>변환된 꽃 그림 갤러리에 저장</li>
</ul>
